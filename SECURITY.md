# Security Policy

## Supported Versions

Security updates are provided for the latest stable release branch/tag.

- Current stable: `v2.0.1`

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Report vulnerabilities privately through Avalon Hosting Services support with:

- Affected version
- Reproduction steps
- Impact description
- Suggested fix (if available)

We will acknowledge receipt and investigate as quickly as possible.
