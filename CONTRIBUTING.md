# Contributing

Thanks for helping improve the Domain Reseller Registrar Module.

## Before You Start

- Check existing issues before creating a new one.
- Open an issue first for major changes.
- Keep pull requests focused on a single topic.

## Development Notes

- Module path must remain: `modules/registrars/domain_reseller_registrar`
- Follow existing coding style and naming patterns.
- Avoid breaking backward compatibility unless clearly discussed.

## Pull Request Checklist

- Code compiles and runs in your WHMCS test setup.
- Changes are documented clearly.
- Any configuration or behavior changes are explained.
- You updated relevant docs when needed.
